# SEO-GEO-STANDARD.md

Use this formula on important pages:

SEO foundation + GEO answer structure + AEO question formatting.

Every important page should:

1. Answer the visitor quickly.
2. Use one clear H1.
3. Use helpful headings.
4. Add short questions and answers when useful.
5. Use simple language.
6. Link to related pages.
7. Keep title and description accurate.
8. Add public evergreen pages to sitemap and llms.txt.

SlotsFreeUSA pages should stay clear, simple, and compliant. Keep claims modest and easy to verify.
